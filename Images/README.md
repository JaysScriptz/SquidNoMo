# SquidNoMo image assets

These PNG files are downloaded at runtime and registered with `getcustomasset`/`getsynasset` when supported.

- `SquidNoMoLogo.png` — app/sidebar/minimized logo
- `BannerGuards.png` — dashboard hero artwork
- `TabIcons/*.png` — navigation tab icons
- `CashAppQR.png` and `PayPalQR.png` — support QR codes
